Cross-Layer Systems
🎯 Pipeline Orchestrator
pipeline-orchestratorSvc
progress-streamerSvc
Cyclic state machine. Manages forward + backward transitions, job queue (BullMQ), cost tracking, SSE progress events.
🔍 Quality Gate
quality-assessorAgent
quality-metricsSkill
Evaluates output at every layer boundary. Three verdicts: PASS, REVISE, REJECT. Max 3 auto-revisions then escalate to user.
🔄 Revision Engine
revision-controllerAgent
version-trackerMCP
Backward transitions to ANY previous layer. Smart cascade: only affected downstream scenes re-generated. Version tracking + diff + revert.
REVISE/REJECT → backward to L1–L4
Cascade → scene-level re-generation
🌐 User Interface
user-bridgeMCP
User ↔ pipeline interface. Submit, review, approve, request revision, override decisions at any checkpoint.
Design Principle: Agents decide what to do (intelligence). MCP Servers execute how (tools). Skills provide knowledge (context). The Orchestrator manages when and in what order. Quality gates verify at every layer boundary. Revision loops enable iterative creative refinement.

sequenceDiagram
  participant U as 👤 User
  participant UB as 🌐 user-bridge
  participant O as 🎯 Orchestrator
  participant A as 🤖 Layer Agent
  participant M as 🔧 MCP Server
  participant Q as 🔍 Quality Gate
  participant R as 🔄 Revision Ctrl
  participant DB as 💾 DB / S3

  U->>UB: Submit project (music, refs, prompt)
  UB->>O: Create pipeline job
  O->>O: Initialize state machine

  loop For each Layer (L1→L5)
    O->>A: Trigger layer agent
    activate A
    Note over A: Skills loaded as context
    A->>M: Call MCP tools
    activate M
    M->>DB: Write versioned output
    M-->>A: Return result
    deactivate M
    A-->>O: Stage complete
    deactivate A

    O->>Q: Evaluate output
    activate Q
    alt PASS
      Q-->>O: Advance to next layer
    else REVISE
      Q->>U: Suggest improvements
      U-->>Q: Approve or revise
    else REJECT
      Q->>R: Auto-retry with feedback
      R->>A: Re-process (targeted scenes)
    end
    deactivate Q
  end

  O->>UB: Pipeline complete
  UB->>U: Deliver final video
